# 覆盖系统可视化工具

## 界面
- **左侧画布**：显示矩形（每个矩形代表一个剩余类）
- **右侧面板**：选择形状、编辑文本、删除元素

## 操作

### 添加矩形
1. 在右侧网格点击一个格子（如 `2,3`）
2. 在画布空白处点击，矩形自动对齐放置

### 选择/移动
- 点击矩形选中（高亮显示）
- 拖动已选矩形移动

### 删除
- 按 `D` 或点击 [D]elete 删除选中矩形
- 点击 [C]lear 删除所有矩形

### 视图控制
- **缩放**：鼠标滚轮
- **平移**：按住 Ctrl/Shift + 拖动鼠标，或拖动边缘滚动条

### 文本编辑
- 下方文本框直接编辑字符串（格式 `t+MZ`）
- 修改后自动更新画布
- **撤销/重做**：Ctrl+Z / Ctrl+Y

### 网格大小
- 用右侧 `+`/`-` 按钮调整可选的形状范围
- 按住 Shift + 右侧面板滚轮也可调整

## 快捷键
| 键 | 功能 |
|----|------|
| D | 删除选中 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Ctrl/Shift+拖动 | 平移画布 |

## 相关构造

一个最小模数为 $2$ 的覆盖系统示例：
```text
0+2Z,0+3Z,1+6Z,1+4Z,11+12Z
```

一个最小模数为 $3$ 的覆盖系统示例：
```text
1+3Z,5+6Z,3+12Z,0+9Z,3+18Z,33+36Z,2+4Z,4+8Z,8+16Z,8+24Z,0+48Z
```

一个最小模数为 $4$ 的覆盖系统示例：
```text
2+4Z,4+8Z,8+16Z,16+32Z,32+64Z,64+128Z,128+256Z,0+192Z,256+384Z,512+768Z,3+6Z,1+12Z,19+24Z,7+48Z,79+96Z,55+72Z,31+144Z,223+288Z,2+9Z,5+18Z,17+36Z,8+27Z,17+54Z,107+108Z
```
